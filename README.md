CYBER COMBAT is a browser-based 2D tactical shooter built using HTML5 Canvas, CSS3, and Vanilla JavaScript. 
When you first open the app, you will see a Login page where you can change your username or password as well as set the options of what type of settings you want to have for your experience using the Settings menu (included in the app). You will be able to log in successfully and see a page with Missions you can select from carefully crafted maps for each mission with varying difficulty levels and obstacle layouts (including but not limited to Training Ground, Urban Warfare, Fortress Assault, Chaos Arena, Final Showdown).
